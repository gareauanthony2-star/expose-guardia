# Analyse de texte en Python

Ce projet contient un script Python (`test.py`) qui analyse un texte fourni par l'utilisateur.  
Il compte le nombre de mots, de lettres, de phrases, et affiche les 5 mots les plus fréquents.

---

## Utilisation

1. Cloner ce dépôt ou télécharger le fichier `test.py`.  
2. Exécuter le script avec Python (version 3.6 ou plus) :  

```power shell
python test.py
